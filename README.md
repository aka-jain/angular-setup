#Angular-Setter
Angular 1.5 setup with bower, SASS, gulp, npm

##Frameworks/Libraries used
1. Angular JS,
2. Jquery
3. Lodash
4. Materialize CSS
5. SASS

##Task Runner/Package Manager
1. Gulp
2. npm
3. bower

##Angular Dependencies
1. Angular 1.5
2. Angular-UI-Router
3. Angular-Animate
4. Angular-Message
5. SmoothScroll

##Bower Dependencies
1. Jquery
2. Lodash
3. Materialize CSS

##Extra Dependencies
1. Font Awesome
2. Animate.css
3. WOW.js (to be included)

##How to start (be ready with node installed on your machine)
```
  npm install
  bower install
  gulp connect
 ```
Server is started at localhost:8888

Now to watch for static files and build dev/prod version run
```
  gulp dev
  gulp watch
```
For production
```
  gulp prod
  gulp watch
```
To configure Environment variables for dev/prod, change config.json file.
